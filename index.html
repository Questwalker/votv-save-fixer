Voices of the Void Save Fixer

This tool provides a collection of fixes for various common issues with VotV save files.

How to use:

  • Select your save file in the file browser button below or drop your save file onto this webpage.
    (Your save files can be found in %localappdata%\VotV\Saved\SaveGames)
  • Select the fixes/patches you want applied with the checkboxes.
  • Click "Fix and download" and let it run. It should only take a couple seconds.
    Once its done, it'll download a fixed save file, which you can drop into your save folder and use.
